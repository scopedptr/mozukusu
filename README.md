# [mozukusu](https://github.com/scopedptr/mozukusu)

このリポジトリでは、「[Mozc](https://github.com/google/mozc) の辞書データを Mozc ではない自作のソフトウェアで処理する人」くらいしか興味を持たないような実験や調査を行っています。

ここにあるすべてのファイルは実験的で、品質、有用性、安定性のいずれもが疑わしいとお考えください。

活動方針は「まずはやってみてから有用性を検証し、有用でないと結論づけたものは削除していく」です。
